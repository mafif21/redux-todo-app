# To Do List App With Redux

## Installation

Install this repo with npm

```bash
  git clone https://github.com/mafif21/redux-todo-app.git
  cd this-project
  npm i
```

## Features

- Create task
- Edit task by id
- Read dummy json database
- Delete task by id
- Add to favorite task

## Tech Stack

**Client:** React, Redux, Bootstrap CDN
